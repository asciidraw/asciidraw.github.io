# Group

The group element is nothing less than a box element with an adjustable group name, describing its' content.

```text

|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
|                                           |
+-------------------------------------------+
```
